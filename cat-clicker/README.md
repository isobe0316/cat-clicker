# 猫あつめクリッカー (Cat Clicker)

かわいい黒猫のクリッカーゲームです。

## 遊び方

1. **猫をクリック**: 画面中央の黒猫をクリック（タップ）すると、「にぼし」がもらえます。
2. **アイテムを購入**: 貯まった「にぼし」を使って、画面下のおみせでアイテムを買うことができます。
   - **猫じゃらし** などのアイテム：クリックした時にもらえる「にぼし」が増えます。
   - **野良猫** などの仲間：1秒ごとに自動で「にぼし」を集めてくれます。
3. **放置して貯める**: 仲間が増えれば、放置しているだけでも「にぼし」がどんどん貯まります。

## 起動方法

```bash
npm install
npm run dev
```

ブラウザで表示されたURL（例: `http://localhost:5173`）を開いてください。

## スマホで遊ぶ方法 (GitHub Pages)

このゲームをスマホで遊ぶには、GitHubにアップロードして公開する必要があります。

1.  **GitHubでリポジトリを作成**:
    *   GitHubにログインし、新しいリポジトリ（例: `cat-clicker`）を作成します。
2.  **コードをプッシュ**:
    *   以下のコマンドをターミナルで実行して、コードをGitHubにアップロードします。
    ```bash
    git remote add origin https://github.com/<あなたのユーザー名>/cat-clicker.git
    git branch -M main
    git push -u origin main
    ```
3.  **公開設定**:
    *   GitHubのリポジトリページで「Settings」→「Pages」を開きます。
    *   「Build and deployment」の「Source」を **「GitHub Actions」** に変更します。
4.  **遊ぶ**:
    *   数分待つと、デプロイが完了します。
    *   表示されたURL（例: `https://<ユーザー名>.github.io/cat-clicker/`）にスマホでアクセスしてください！
